# Contributing

Thanks for your interest!

This is a small educational project. If you want to contribute:

1) Fork the repo
2) Create a branch:
   - `git checkout -b feature/my-change`
3) Run tests:
   - `pytest`
4) Run format + lint:
   - `ruff format .`
   - `ruff check .`
5) Open a Pull Request with:
   - what changed
   - why it changed
   - how to reproduce

Please keep the model assumptions **explicit** and the code **reproducible**.
